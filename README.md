# pattern_Observer
pattern_Observer from book Erik Friman Patterns of Programming


  Пример поведентческого патеерна проектирования "Наблюдатель" ("Observer").
  
  Работает по модели "Подписчик" - "Издатель". 
Издатель регистрирует подписчиков, а затем рассылает им информацию. 
Пример из реальной жизни - подписка на какие-либо газеты, журналы. 

Пример взят из книги "Паттерны проектирования" Эрика и Элизабеты Фриман.

Также использовался материал сайта https://refactoring.guru/ru/design-patterns/observer
